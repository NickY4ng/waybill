## 四、项目启动步骤

### 4.1 克隆仓库

```bash
# 使用SSH克隆（推荐）
git clone git@github.com-nicky4ng:NickY4ng/waybill.git

# 或者使用HTTPS
git clone https://github.com/NickY4ng/waybill.git
```

### 4.2 安装依赖

```bash
cd waybill/app
npm install
```

### 4.3 启动开发服务器

```bash
npm run dev
```

### 4.4 构建生产版本

```bash
npm run build
```

## 六、重要注意事项

### 6.1 API密钥安全

- 所有API Key都硬编码在源代码中
- 生产环境部署时建议使用环境变量管理敏感信息
- 不要将包含API Key的文件提交到公开仓库

### 6.2 SSH密钥

- 确保新电脑上配置了正确的SSH密钥
- 如果使用不同的SSH密钥文件名，需要更新 `~/.ssh/config`

### 6.3 本地存储

项目使用 `localStorage` 存储以下数据：
- 百炼会话ID: `bailian_session_id`
- 深度分析会话ID: `deep_analysis_session_id`
- 消息计数
- 分析报告缓存
